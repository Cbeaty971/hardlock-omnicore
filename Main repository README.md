# HARD‑LOCK Multi‑Firm Agent Architecture

A unified, local‑only, multi‑agent legal system designed for document analysis, drafting, research, and procedural automation. This repository contains:

- System prompts  
- Multi‑agent architecture  
- UI (Agent.html + CSS + JS)  
- Document‑processing engines  
- Diagnostics  
- SYPHER Protocol  
- Master Document Agent  

All components are designed to operate under HARD‑LOCK constraints (local‑only, no APIs, no external calls).
