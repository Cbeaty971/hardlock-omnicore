# hardlock-omnicore
HardLock Omnicore is a unified local‑only multi‑agent legal intelligence system for document analysis, drafting, research, diagnostics, and procedural automation. Enforces HARD‑LOCK constraints, uses multi‑model reasoning, and includes a Times New Roman UI with modular agents and processing engines.
